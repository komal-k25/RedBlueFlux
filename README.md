# RedBlueFlux 

An interactive browser-based hand tracking experience built with **HTML, CSS, JavaScript, and MediaPipe**.
Uses real-time webcam detection to track both hands and generate glowing elastic energy bands between matching fingertips.

## ✨ Features

* 🖐️ Real-time hand tracking using webcam
* 🔴🔵 Red and blue dual-hand visual theme
* ⚡ Elastic glowing bands between fingertips
* 🎥 Smooth interactive browser experience
* 🎨 Cinematic visuals and responsive effects
* 🌐 Runs directly in the browser (no installation required)

## 🚀 Live Demo

```text
https://komal-k25.github.io/RedBlueFlux/
```

## 💡 How It Works

* Detects both hands in real time using MediaPipe
* Tracks fingertip positions
* Connects matching fingertips with animated elastic bands
* Uses Canvas rendering for glowing visual effects

## ▶️ Run Locally

* Download or clone this repository
* Open index.html in your browser
* Allow webcam access
* Show both hands and interact
